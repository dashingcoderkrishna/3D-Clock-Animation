# 3D Clock Project (Maya + Python)

This project contains a fully functional 3D analog clock created in Autodesk Maya.  
The clock hands rotate in real-time using a Python script that syncs with the system clock.

## Features
- Real-time hour, minute, and second hand rotation
- Pivot-based animation setup
- Python script using scriptJob for continuous updates
- Clean Maya hierarchy
